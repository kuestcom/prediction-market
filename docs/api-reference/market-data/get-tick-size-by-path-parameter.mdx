---
title: Get Tick Size By Path Parameter
description: "Get minimum tick size."
_openapi:
  method: GET
---
Returns the minimum tick size enforced for the requested token.

Tick sizes are returned as numbers. The current default minimum is `0.01` unless otherwise configured.
