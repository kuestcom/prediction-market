---
title: Get Midpoint Prices Request Body
description: "Get midpoints for many tokens."
_openapi:
  method: POST
---
Retrieve midpoint prices for multiple tokens in a single request.

The response is a map where each key is the requested `token_id` and the value is the midpoint price.
