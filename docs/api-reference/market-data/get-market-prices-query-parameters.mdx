---
title: Get Market Prices Query Parameters
description: "Get the best price for a side."
_openapi:
  method: GET
---
Retrieves the latest executable price for a specific `token_id` on the indicated side of the order book.

Switch the `side` query parameter to `SELL` to retrieve the best ask for the same outcome.
