---
title: Get Midpoint Price
description: "Get the bid/ask midpoint."
_openapi:
  method: GET
---
Returns the midpoint between the best bid and best ask for a specific market outcome.

The `mid` price is returned as a string so that clients can preserve precision when parsing the response.
