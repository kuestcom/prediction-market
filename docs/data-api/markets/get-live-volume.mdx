<code>GET {ENV.DATA_URL ?? The response contains the overall `total` plus a `markets` array that breaks the number out per condition ID.

Use it for dashboards that highlight the hottest events or for alerting tools that watch for sudden bursts of liquidity. The required `market` query expects a comma-separated list of condition IDs; pass every condition that belongs to the event to scope the snapshot correctly.
