---
title: 'Get multiple order books summaries by request'
openapi: 'POST /books'
---


Submit a batch of `token_id`s to retrieve their latest order book snapshots in a single call.

<Details title="View example request">

```bash
curl -X POST "${CLOB_URL:-https://clob.kuest.com}/books" \
  -H 'Content-Type: application/json' \
  -d '[{"token_id":"66991175107696133528353695394151645462081911990365106169948013117379189357520","side":"BUY"},{"token_id":"106199154714545451005463800266480893448383896699905306895283360115694583822185","side":"SELL"}]'
```

</Details>
