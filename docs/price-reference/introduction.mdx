---

The Price Reference API exposes canonical mark windows for live-series markets.

Base URL: `https://price-reference.kuest.com`

Use this service to:

- Fetch the latest finalized mark per instrument/interval in one request.
- Read historical finalized mark windows.
- Resolve `series_slug` to instrument metadata.
- Trigger an immediate ingest pass (internal endpoint).
